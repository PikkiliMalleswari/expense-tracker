# Expense Tracker

**Expense Tracker** is a Python (Flask) web app that helps users manage personal finances. Users can add, view, and delete expenses, track spending by category and month, receive budget alerts, and export expenses as CSV. It provides a simple and interactive way to monitor daily spending efficiently.

---

## Features
- Add, view, and delete expenses
- Display total spending dynamically
- Category-wise expense summary
- Monthly spending report
- Budget alert when exceeding the set limit
- Export expenses as CSV

---

## Technologies Used
- Python (Flask)
- HTML
- CSS
